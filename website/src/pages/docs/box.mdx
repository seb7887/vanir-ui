# Box

Box is the most abstract component on top of which all other components are built.

By default, it renders a _div_ element.

## Import

```tsx
import { Box } from 'vanir-ui'
```

## Usage

<Example>
  <Box className="bg-red-400 p-2 text-white">This is a Box</Box>
</Example>

```tsx
<Box className="bg-red-400 p-2 text-white">This is a Box</Box>
```
